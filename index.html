Como nós Trabalhamos

Colocamos seu projeto no ar em apenas 3 etapas

01 - Análise

Esse processa visa em coletar informações da aplicação e requisitos funcionais.

Com base nisso, iniciar a conversação para o layout da aplicação.

Implementação do layout rascunho.

Após finalização do rascunho (layout da aplicação) será enviado para avaliação para o usuário final para a aprovação do mesmo.

Após a aprovação do usuário final se inicia a etapa 2.

02 - Desenvolvimento do Projeto

Com base em toda a análise da primeira etapa será definido um prazo para o desenvolvimento da aplicação.

No meio do desenvolvimento será repassado ao cliente uma prévia para o mesmo acompanhar o progresso do desenvolvimento.

03 - Publicação do Projeto

Após o término do desenvolvimento iremos passar novamente pela última revisão da aplicação junto com o cliente.

Após a aprovação do mesmo será colocado a aplicação no ar.

* Todo o processo listado acima será documentado nas cláusulas do contrato.
